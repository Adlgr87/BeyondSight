# ⚙️ Motores de Simulación

BeyondSight incluye dos motores matemáticos complementarios. Ambos comparten rango de opinión, métricas de salida y estructura de historial, pero difieren en paradigma y caso de uso.

## 📐 Motor Discreto (`simulator.py`)

- **Base**: Modelos académicos de formación de opinión (Friedkin-Johnsen, Hegselmann-Krause, Granovetter, Axelrod, EGT).

- **Mecánica**: Reglas de transición seleccionadas
